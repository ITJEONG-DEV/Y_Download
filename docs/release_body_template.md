### ✨ 주요 기능
- URL 조회 → 제목·썸네일·길이 표시
- 여러 영상을 **목록에 담아 한 번에 다운로드**
- 항목별 **파일명·확장자·포맷(영상/음원)·품질** 개별 설정
- 선택 옵션 기준 **예상 파일 크기** 표시
- **다운로드 내역**(성공/실패) 기록·조회, 더블클릭으로 목록에 재추가
- 마지막 **저장 위치 기억**
- 실행 시 **자동 업데이트 확인**

### 📦 어떤 걸 받아야 하나요? (full vs lite)
| | **full (권장)** | **lite** |
|---|---|---|
| 파일 | `Y_Downloader-full-{{VERSION}}.zip` | `Y_Downloader-lite-{{VERSION}}.zip` |
| ffmpeg | ✅ 포함 (바로 실행) | ❌ 미포함 (별도 준비) |
| 용량 | 약 100MB | 약 27MB |
| 사용법 | 압축 풀고 `Y_Downloader.exe` 실행 | exe 옆에 `ffmpeg.exe`를 두거나 시스템에 설치 |
| 추천 대상 | 대부분의 사용자 | ffmpeg가 이미 있거나 용량을 아끼려는 사용자 |

> ffmpeg는 **영상+음성 합치기, mp3 변환**에 필요합니다. lite 사용 시 동봉된 `lite-ffmpeg-안내.md`를 참고하세요.
